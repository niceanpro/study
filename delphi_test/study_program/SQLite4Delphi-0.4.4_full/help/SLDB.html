Types

NameDescription
TCBDBFlag callback flag.

Callback is a function called by the engine - this flag determines what operation is pendning:

  • cbfUnknown: default, just to allow "empty" value.
  • cbfDBList: list of databases.
  • cbfTblList: table list.
  • cbfIdxList: list of indices.
  • cbfInternal: internal data - not used.
