Class TFunArgManager

type TFunArgManager = class()

Description

Manager allows callback to determine which database is used for the function / aggregation

Next UserData can be retrived, since callback knows the DB. Right now there is no need to do that - manager returns pointer to TSLFunction object. It can be declared in implementation section since the only variable __FunArgManager of this class is declared there. But in that case PasDoc would not produce this help.
